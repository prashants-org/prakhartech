# Prakhar Technology Website (React)

This project converts the original static HTML/CSS site into a React application using Vite.

## Run locally

1. Install dependencies:
   - npm install
2. Start the dev server:
   - npm run dev
3. Build for production:
   - npm run build

## GitHub Pages deployment

This repo includes a GitHub Actions workflow that builds and deploys to GitHub Pages on every push to `main` or `master`.

1. In GitHub, go to **Settings â†’ Pages**.
2. Set **Build and deployment** to **GitHub Actions**.
3. Push to `main` (or `master`) and the site will publish automatically.
